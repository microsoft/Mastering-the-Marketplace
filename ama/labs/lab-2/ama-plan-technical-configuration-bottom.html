More settings are available to you for the technical configuration of this plan.

  1. The Azure AD tenant ID for the publisher.
  2. Authorizations, which are IDs of users or services that should have access to the items in the managed resource group.
  3. Policy settings for items in the managed resource group. This uses Azure policies.

The technical configuration screen allows the following actions (bottom of page).

  1. Deployment mode (complete or incremental).
  2. Adding a webhook URL for events fired by the Managed App provisioning system.
  3. Customizing allowed actions on the resources in the Managed Resource Group that will be created.
  4. Add the publisher's Tenant ID.
  5. The Principal ID of users or other resources (like an Azure function) that will have access to the resources created in the Managed Resource Group.
  6. Policy settings that you may want to apply to the managed environment.

Click near the top of the screen to scroll up and see more of the Technical configuration page.

Click the "Review and Publish" button at the bottom of the page to review your submission.
